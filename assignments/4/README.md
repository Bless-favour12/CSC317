# Assignment 4 — JavaScript Portfolio

## Overview
This assignment introduces basic JavaScript functionality by connecting HTML, CSS, and JS files together in a simple webpage.

## Files
- `index.html`: Main webpage that links CSS and JavaScript.
- `styles/styles.css`: Contains blue-themed styling for the webpage.
- `scripts/hello.js`: Displays a greeting alert when the user clicks the button.
- `images/`: Placeholder folder for future assets.

## How to Run
1. Open `index.html` in a web browser.
2. Click the “Click Me” button to see the JavaScript alert message.
3. View the source code to explore the link between HTML, CSS, and JS.
